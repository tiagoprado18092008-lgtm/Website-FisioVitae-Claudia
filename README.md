# Fisiovitae - Website

Website profissional para clínica de fisioterapia Fisiovitae.

## 📋 Estrutura do Projeto

```
fisiovitae/
├── index.html              # Página inicial
├── blog.html               # Blog
├── contacto.html           # Contato
├── equipa.html             # Equipa
├── 404.html                # Página de erro
├── assets/                 # Arquivos estáticos
│   ├── css/               # Estilos
│   ├── js/                # Scripts
│   └── images/            # Imagens
├── pages/                  # Páginas específicas
│   ├── fisioterapia-ecoguiada.html
│   └── fisioterapia-percutanea.html
└── README.md
```

## 🎯 Funcionalidades

- Homepage com seção hero
- Página de blog
- Página de contato
- Página de equipa
- Páginas de serviços específicos
- Design responsivo
- Performance otimizada

## 🚀 Como Usar

1. Clone o repositório
2. Abra os arquivos HTML no navegador
3. Para desenvolvimento, use um servidor local (ex: `python -m http.server`)
